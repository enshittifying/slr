## CRITICAL RULES TO REMEMBER

### ✓ DO:
- Use abbreviations **only in citations**
- Follow **exact table format** for spacing and periods
- Use **Table T10** for geographic terms
- Use **Table T6** for case name words
- Use **Table T7** for court names
- Check **whether adjectives** need spelling out
- Include **court designation in parentheses** when not obvious

### ✗ DON'T:
- Create abbreviations not in Bluebook tables
- Abbreviate in textual sentences (except widely known acronyms)
- Abbreviate **geographic adjectives** (spell out "California," not "Cal.")
- Use wrong reporter abbreviation variants
- Forget periods in abbreviations
- Mix abbreviation styles
- Use "Uniform" instead of "Unif."

## SPECIAL SITUATIONS

### Geographic Adjectives
Always spell out - never abbreviate!
```
❌ WRONG: "The Cal. statute..."
✓ RIGHT: "The California statute..."
```

### Ampersand (&)
Use ONLY in official case names
```
✓ RIGHT: AT&T Corp. v. Sprint Communications Co.
✓ RIGHT: John & Mary Doe (if official name)
❌ WRONG: "The statute & case law both show..."
```

### Widely Recognized Acronyms
Can appear in text without periods
```
✓ RIGHT: "The FBI and CIA coordinated..."
✓ RIGHT: "NAFTA regulations apply..."
❌ WRONG: "The bureau & agency both agreed..."
```
