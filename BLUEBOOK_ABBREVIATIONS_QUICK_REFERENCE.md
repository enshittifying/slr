### EXPLANATORY PHRASES

**Affirmation**
```
aff'd              affirmed
aff'g              affirming

```

**Reversal**
```
rev'd              reversed

```

**Certiorari**
```

cert. denied       Supreme Court refused review

```

**Modification**
```

```

## QUICK CHECKLIST FOR ABBREVIATIONS

**When abbreviating:**
- [ ] Check Table T6 for case name words
- [ ] Check Table T1 for reporters and statutes
- [ ] Check Table T7 for court names
- [ ] Check Table T10 for geographic terms
- [ ] Use proper periods and spacing
- [ ] Preserve apostrophes where indicated (e.g., Att'y, Sec'y)
- [ ] Don't create abbreviations that aren't in the tables
- [ ] For geographic adjectives, spell out (e.g., "Californian," not "Cal.")
- [ ] Spell out words in textual sentences unless universally known
- [ ] Always use "Unif." for Uniform Laws

---

## COMMON MISTAKES TO AVOID

❌ Creating non-standard abbreviations
✓ Using only abbreviations from Bluebook tables

❌ Abbreviating geographic adjectives (e.g., "Cal." for "Californian")
✓ Spelling out geographic adjectives

❌ Abbreviating in textual sentences
✓ Abbreviating only in citations

❌ Inconsistent spacing in reporter citations
✓ Using proper spacing (e.g., "F.3d" not "F. 3d")

❌ Omitting periods in abbreviations
✓ Including periods as shown in tables

❌ Using "Uniform" instead of "Unif."
✓ Always use "Unif." for uniform laws

❌ Mixing abbreviation styles
✓ Use consistent style throughout document
