# Rule 12: Statutes - Comprehensive Guide with Regex Patterns

## Complete Rule 12 Hierarchy

### Rule 12 (Main)
**Title:** Statutes

**Text:** Cite statutes by their official or popular name (if commonly used) and the published source (code or session law) and year. Federal statutes: cite to the U.S. Code when available: Title U.S.C. § Section (Year). E.g., 42 U.S.C. § 1983 (2018). For state statutes: cite to the current code if available or the session law if not codified. Include the jurisdiction abbreviation if not clear (e.g., "Va. Code Ann."). If citing an unofficial code, indicate the publisher and year parenthetically. Include supplements if applicable ("Supp. 2020").

---

## Sub-Rules and Content

### Rule 12.1: Basic Citation Forms
**Text:** The basic form for statutes: [Official Name of Act], [published source] [section symbol and number] ([year]). If the act is commonly known by a name, you can use that name in italics (for academic writing). Federal code citation: [Title] U.S.C. § [section] ([Year]). State code citation: [State Abbrev.] [Code Abbrev.] § [section] ([Year]). Session law: [Act Name], [Pub. L. or Chapter] No. [number], [volume] [Reporter] [page] ([Year]).

**Key Examples:**
- Federal: 42 U.S.C. § 1983 (2018)
- State: Va. Code Ann. § 8.01-273
- Session Law: Civil Rights Act of 1964, Pub. L. No. 88-352, 78 Stat. 241 (1964)

---

### Rule 12.2: Choosing the Proper Citation Form
**Text:** Use the official code if available (e.g., U.S.C. for federal statutes). If a statute is spread across multiple code sections, cite as a range (e.g., 42 U.S.C. §§ 9601-9675). If the statute is not in the code yet, cite the session law. If an unofficial code is more up-to-date or commonly used in practice, you may use it in court documents (with an indication of currency or year). In law review, typically stick to official codes whenever possible.

#### Rule 12.2.1: General Rule
**Text:** Cite to the current official code (or its supplement) if available; otherwise, cite an unofficial code or the session laws, in that order of preference. Ensure that whichever source is used will allow the reader to find the law easily.

#### Rule 12.2.2: Exceptions
**Text:** In certain circumstances, such as when a statute has been repealed or amended and those changes are relevant, it may be appropriate to cite an earlier version or to a source discussing those changes (like a legislative history or a prior code edition). Another exception is if a jurisdiction requires parallel citation to an unofficial code, in which case follow the local rules.

---

### Rule 12.3: Current Official and Unofficial Codes
**Text:** Federal: The official code is the United States Code (U.S.C.), published every six years with supplements. Unofficial codes include U.S.C.A. and U.S.C.S. For state codes, most states have an official code; some also have unofficial annotated codes. Always cite the official code if up-to-date; if using an unofficial code for a recent amendment, indicate it. Include the year of the code edition (or supplement) used in parentheses. If citing an unofficial code in a court document, indicate the publisher abbreviation (e.g., West or Lexis) if required by local rules.

#### Rule 12.3.1: Additional Information
**Text:** If the code edition year and the actual text year differ, the Bluebook typically wants the edition year (which is usually on the spine of the volume or the title page). If citing a pocket part or supplement, use the year on that supplement. Some codes also have effective dates or credit information—generally do not include those in citations, but be mindful if a provision's effectiveness is at issue.

#### Rule 12.3.2: Year of Code
**Text:** When citing the U.S. Code, you no longer need to include a date if the code is current (Bluebook 21st edition changed this rule—federal code citations omit the year if citing the current code). For state codes, include the year of the code volume or supplement if the publisher prints one; if citing an electronic version, use the currency date given.

**Key Points:**
- U.S.C. (official): 42 U.S.C. § 1983 (year optional if current)
- U.S.C.A. (unofficial, West): 42 U.S.C.A. § 1983 (2020)
- U.S.C.S. (unofficial, Lexis): 42 U.S.C.S. § 1983 (2020)

---

### Rule 12.4: Session Laws
**Text:** Cite session laws when a statute is not yet in the code or if you want to refer to the law as originally passed. Format: [Official Name of Act], [Pub. L. No. or Chapter] [No.], [Volume] [Reporter Abbrev.] [Page] ([Year]). E.g., Civil Rights Act of 1964, Pub. L. No. 88-352, 78 Stat. 241 (1964). For state session laws, use the state abbreviation and the session law citation format (often year and chapter).

**Key Examples:**
- Federal: Civil Rights Act of 1964, Pub. L. No. 88-352, 78 Stat. 241 (1964)
- State: Act of May 15, 2020, ch. 302, 2020 Ill. Stat. 555

---

### Rule 12.5: Electronic Databases and Online Sources
**Text:** When a statute is found only online or in a database (or if that is the source being cited), give a parenthetical indicating the database and currency. For example: 42 U.S.C. § 1983 (LEXIS through Pub. L. No. 116-140). If citing a state code online, follow local conventions for currency. Use authenticated PDFs or official websites when possible, and provide a URL if required or if it aids in retrieval (Bluebook rule 18.2.2 might apply as well).

---

### Rule 12.6: Other Secondary Sources
**Text:** Occasionally you may need to cite a secondary source for statutory material (like a looseleaf, a treatise, or an agency guidance that includes statutory text). Cite those like you would any similar secondary source (rule 15 or 18, etc.), but make clear what part is being cited. It's usually better to cite the statute directly if possible.

---

### Rule 12.7: Invalidation, Repeal, Amendment, and Prior History

#### Rule 12.7.1: Invalidation
**Text:** If a statute or part thereof has been held invalid (e.g., unconstitutional), and that information is relevant, include a parenthetical in the citation indicating the disposition.

**Example:** 18 U.S.C. § 3501 (2000) (held unconstitutional by Dickerson v. United States, 530 U.S. 428 (2000))

#### Rule 12.7.2: Repeal
**Text:** If a statute has been repealed, note the repeal year in the citation if citing an historical version. If citing current code, you wouldn't include a repealed section unless you specifically discuss its prior existence, in which case use an explanatory parenthetical.

**Example:** 42 U.S.C. § 6000 (1988) (repealed 1992)

#### Rule 12.7.3: Amendment
**Text:** If a recent amendment is relevant, you might include the session law of the amendment in a parenthetical or as a separate citation. Generally, if citing the current code, you don't indicate it was amended unless necessary (the code text is presumed current). If discussing a forthcoming amendment, you could cite the public law number that enacted it, as pending or as a reference.

#### Rule 12.7.4: Prior History
**Text:** For acts with complicated histories, you may cite to earlier versions or note former classifications if relevant (e.g., if an act was renumbered or moved in the code). This is rarely needed in standard citation but could appear in historical discussions.

---

### Rule 12.8: Explanatory Parenthetical Phrases
**Text:** When citing statutes, you might add explanatory parentheticals to clarify how the statute relates to your proposition. E.g., (providing definitions) or (establishing regulatory authority for...). These follow any prior history parentheticals and are similar to case parentheticals in form.

---

### Rule 12.9: Special Citation Forms

#### Rule 12.9.1: Internal Revenue Code
**Text:** Cite the Internal Revenue Code (Title 26 of U.S.C.) by section alone, but often precede with "I.R.C." for clarity in textual discussions. E.g., I.R.C. § 61. In academic writing, 26 U.S.C. § 61 (2018) is acceptable, but practitioners often just use I.R.C. for brevity when it's understood the reference is to the tax code.

**Forms:**
- I.R.C. § 61
- 26 U.S.C. § 61 (2018)
- I.R.C. § 1366(a)(1)

#### Rule 12.9.2: Ordinances
**Text:** Cite municipal ordinances by the name of the municipality and the ordinance (or code) section. Include the jurisdiction to avoid confusion (city and state). If the ordinance is from a code, specify it's a code. If it's a one-off ordinance not codified, cite the ordinance number and date of passage.

**Example:** Chicago, Ill., Municipal Code § 8-4-015 (2020)

#### Rule 12.9.3: Rules of Evidence and Procedure
**Text:** Cite federal rules as: Fed. R. [Civ./Crim./Evid./App.] P. [Rule number]. State rules similarly: [State Abbrev.] R. Civ. P. [number], or just Rule [number] if context makes clear. For local court rules, include the court name: e.g., S.D.N.Y. R. 4.1. Use "R." for Rule in textual sentences if starting a sentence, otherwise abbreviate in citations.

**Examples:**
- Fed. R. Civ. P. 56
- Fed. R. Crim. P. 26
- Fed. R. Evid. 801
- Fed. R. App. P. 32
- Cal. R. Ct. 8.1012
- S.D.N.Y. R. 4.1
- N.Y. R. Civ. P. 213

#### Rule 12.9.4: Model Codes, Principles, Restatements, Standards, Sentencing Guidelines, and Uniform Acts
**Text:** Cite these by name and section if available. Include year of the source. If citing the unofficial version of the UCC as codified, use that citation instead.

**Examples:**
- Model Penal Code § 210.3 (Am. Law Inst. 1962)
- Restatement (Second) of Contracts § 90 (Am. Law Inst. 1981)
- Uniform Commercial Code § 2-201 (AM. LAW INST. & UNIF. LAW COMM'N 2017)
- U.S. Sentencing Guidelines § 4A1.1 (U.S. Sentencing Comm'n 2016)

#### Rule 12.9.5: ABA Code of Professional Responsibility and Opinions on Ethics
**Text:** Cite the ABA Model Rules of Professional Conduct or Code of Professional Responsibility by rule or DR number, and year if needed. For ethics opinions, include the organization, opinion number, and year.

**Examples:**
- Model Rules of Pro. Conduct r. 1.7 (Am. Bar Ass'n 2020)
- ABA Comm. on Ethics & Pro. Resp., Formal Op. 466 (2014)

---

### Rule 12.10: Short Forms for Statutes
**Text:** After giving one full citation of a statute, you can shorten subsequent citations. Options: Use "Id." if the immediately preceding footnote is the same statute and same section. Otherwise, you can use an abbreviation like the section number and code (e.g., § 1983) if context makes it clear which code is being referenced (with a prior cite to 42 U.S.C. § 1983, you might later just say § 1983, assuming no confusion). Do not use "supra" for statutes; use the short name or abbreviation instead.

---

## Comprehensive Regex Patterns for Statute Citations

### 1. Federal Code Citations

#### Pattern 1.1: Standard Federal Code (U.S.C.)
**Correct Form:**
```regex
\b\d{1,3}\s+U\.S\.C\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s*\(\d{4}\)?
```

**Description:** Captures federal code citations in the format [Title] U.S.C. § [section] ([Year])

**Correct Examples:**
- 42 U.S.C. § 1983 (2018)
- 5 U.S.C. § 552
- 28 U.S.C. § 1331 (2020)

**Incorrect Patterns to Avoid:**
```regex
# Missing section symbol
\b\d{1,3}\s+U\.S\.C\.\s+\d+(?:\([A-Za-z0-9\s\-\.]+\))?

# Wrong code abbreviation
\b\d{1,3}\s+US\.C\.\s+§+\s*\d+

# Missing title number
\bU\.S\.C\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s*\(\d{4}\)?
```

---

#### Pattern 1.2: Federal Code with Range
**Correct Form:**
```regex
\b\d{1,3}\s+U\.S\.C\.\s+§§\s*\d+(?:\s*[-–]\s*\d+)(?:\s*,\s*\d+(?:\s*[-–]\s*\d+))*\s*\(\d{4}\)?
```

**Description:** Captures ranges of federal code sections

**Correct Examples:**
- 42 U.S.C. §§ 9601-9675 (2010)
- 5 U.S.C. §§ 551-559 (2018)
- 28 U.S.C. §§ 1331, 1332 (2020)

---

#### Pattern 1.3: Federal Code with Supplement
**Correct Form:**
```regex
\b\d{1,3}\s+U\.S\.C\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s+\(Supp\.\s+\d{1,2}\s*\)?\s*\(\d{4}\)?
```

**Description:** Captures federal code citations with supplement notation

**Correct Examples:**
- 42 U.S.C. § 1983 (Supp. 2020)
- 5 U.S.C. § 552 (Supp. III 2020)

---

#### Pattern 1.4: Unofficial Federal Code (U.S.C.A. or U.S.C.S.)
**Correct Form:**
```regex
\b\d{1,3}\s+U\.S\.C\.[AS]{1,2}\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s*\(\d{4}\)?
```

**Description:** Captures unofficial federal code citations from West (U.S.C.A.) or Lexis (U.S.C.S.)

**Correct Examples:**
- 42 U.S.C.A. § 1983 (West 2020)
- 42 U.S.C.S. § 1983 (LexisNexis 2020)
- 5 U.S.C.A. § 552 (West Supp. 2021)

---

#### Pattern 1.5: Federal Code with Database Currency
**Correct Form:**
```regex
\b\d{1,3}\s+U\.S\.C\.\s+§+\s*\d+\s*\([A-Z]{4,}\s+through\s+Pub\.\s+L\.\s+No\.\s+\d{3}[-–]\d{1,3}\)
```

**Description:** Captures federal code with database currency notation

**Correct Examples:**
- 42 U.S.C. § 1983 (LEXIS through Pub. L. No. 116-140)
- 5 U.S.C. § 552 (Westlaw through Pub. L. No. 117-58)

---

## Incorrect Citation Patterns to AVOID

### Federal Code Errors

**Error Pattern 1: Missing section symbol**
```
42 USC 1983 (2018)  ✗ INCORRECT
42 U.S.C. 1983 (2018)  ✗ INCORRECT
```
**Correct:** 42 U.S.C. § 1983 (2018)

---

**Error Pattern 2: Wrong abbreviation**
```
42 US.C. § 1983 (2018)  ✗ INCORRECT
42 USA § 1983 (2018)  ✗ INCORRECT
```
**Correct:** 42 U.S.C. § 1983 (2018)

---

**Error Pattern 3: Space issues in abbreviation**
```
42 U S C § 1983 (2018)  ✗ INCORRECT
42U.S.C. § 1983 (2018)  ✗ INCORRECT
```
**Correct:** 42 U.S.C. § 1983 (2018)

---

**Error Pattern 4: Using outdated citation form**
```
§ 1983, 42 U.S.C. (2018)  ✗ INCORRECT (section before title)
```
**Correct:** 42 U.S.C. § 1983 (2018)

---

**Error Pattern 5: Incorrect range separator**
```
42 U.S.C. §§ 9601-9675  ✗ POTENTIALLY INCORRECT (depends on context - should include year)
```

## Citation Patterns by Statute Category

### 1. Federal Statutes
**Hierarchy of citation preference:**
1. U.S.C. (Official Code)
2. U.S.C.A. or U.S.C.S. (Unofficial Codes - if more current)
3. Statutes at Large (Session Laws - if not yet codified)

**Example citations:**
```
42 U.S.C. § 1983 (federal statute - criminal procedure)
28 U.S.C. § 1331 (federal jurisdiction)
5 U.S.C. § 552 (FOIA)
I.R.C. § 61 (Internal Revenue Code)
Civil Rights Act of 1964, Pub. L. No. 88-352, 78 Stat. 241 (1964)
```

---

### 2. State Statutes
**Key characteristics:**
- Must include state abbreviation
- Include code name (Code, Revised Statutes, etc.)
- Include year in parentheses
- Include publisher if unofficial

**Example citations:**
```
Ala. Code § 6-5-20 (2020)
Cal. Penal Code § 187 (West 2018)
Fla. Stat. § 776.031 (2021)
N.Y. Gen. Bus. Law § 349 (McKinney 2020)
Tex. Bus. & Prof. Code Ann. § 301.001 (Vernon 2019)
Va. Code Ann. § 8.01-273 (2018)
```

---

### 3. Federal Rules
**Key characteristics:**
- Must use abbreviations (Fed. R. Civ. P., Fed. R. Evid., etc.)
- Do NOT include year (rules are presumed current)
- Use subsection notation when appropriate

**Example citations:**
```
Fed. R. Civ. P. 56
Fed. R. Crim. P. 26
Fed. R. Evid. 801
Fed. R. App. P. 32
```

---

### 4. Court Rules
**Key characteristics:**
- Include court designation
- Include rule number and subsection if applicable
- May include court abbreviation

**Example citations:**
```
S.D.N.Y. R. 4.1 (Southern District of New York)
Cal. R. Ct. 8.1012 (California Rules of Court)
N.D. Ill. R. 1.4 (Northern District of Illinois)
```

---

### 5. Ordinances
**Key characteristics:**
- Must include full municipality name
- Must include state abbreviation
- Include "Code" designation
- Include year

**Example citations:**
```
Chicago, Ill., Municipal Code § 8-4-015 (2020)
New York, N.Y., City Code § 1-101 (2018)
San Francisco, Cal., Administrative Code § 1.1 (2019)
Los Angeles, Cal., Municipal Code § 14.00 (2019)
```

---

### 6. Model Codes and Restatements
**Key characteristics:**
- Include edition (First, Second, Third, Fourth)
- Include section symbol
- Include author/organization and year in parentheses

**Example citations:**
```
Model Penal Code § 210.3 (Am. Law Inst. 1962)
Restatement (Second) of Contracts § 90 (Am. Law Inst. 1981)
Restatement (Third) of Torts § 7 (Am. Law Inst. 2010)
Uniform Commercial Code § 2-201 (AM. LAW INST. & UNIF. LAW COMM'N 2017)
```

## Special Cases and Context-Dependent Rules

### 1. Year/Date Requirements
**Current Official Code:**
- U.S.C.: Year optional if current
- State codes: Year typically required
- Rules: No year needed (presumed current)

**Historical References:**
- Always include year for prior versions
- Include year for repealed statutes
- Include year for amended versions when relevant

**Format Examples:**
```
42 U.S.C. § 1983  (current federal code - year optional)
42 U.S.C. § 1983 (2018)  (with year for clarity)
Cal. Code § 1234 (2020)  (state code - year required)
42 U.S.C. § 6000 (1988) (repealed 1992)  (historical version)
```

---

### 2. Supplement Citations
**Format:** `[Citation] (Supp. [Year])`

**Examples:**
```
42 U.S.C. § 1983 (Supp. 2020)  (federal statute with pocket part)
Tex. Code § 301 (Supp. 2021)  (state statute with supplement)
42 U.S.C. § 1983 (Supp. I 2020)  (first supplement)
42 U.S.C. § 1983 (Supp. III 2020)  (third supplement)
```

---

### 3. Parentheticals for Context
**Valid explanatory parentheticals:**
```
42 U.S.C. § 1983 (providing civil remedy for constitutional violations)
Cal. Penal Code § 187 (defining murder)
28 U.S.C. § 1331 (establishing federal question jurisdiction)
I.R.C. § 61 (defining gross income)
```

---

### 4. Invalid Disposition Citations
**Example of correct format:**
```
18 U.S.C. § 3501 (2000) (held unconstitutional by Dickerson v. United States, 530 U.S. 428 (2000))
```

---

### 5. Short Forms
**Allowed forms after full citation:**
```
Full: 42 U.S.C. § 1983 (2018)
Short: § 1983 (if context is clear)
Short: Id. (if immediately preceding citation)
```

## Complete Regex Reference Implementation

Here's a comprehensive Python implementation using all the patterns:

```python
import re

class StatuteCitationValidator:
    """Validate and parse statute citations according to Bluebook Rule 12"""

    def __init__(self):
        """Initialize regex patterns for various statute citation formats"""

        # Federal Code Patterns
        self.federal_usc = re.compile(
            r'\b\d{1,3}\s+U\.S\.C\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s*\(\d{4}\)?',
            re.IGNORECASE
        )

        self.federal_usca = re.compile(
            r'\b\d{1,3}\s+U\.S\.C\.A\.\s+§+\s*\d+(?:\([A-Za-z0-9\s\-\.]+\))?\s*\((?:West|Lexis)\s+\d{4}\)?',
            re.IGNORECASE
        )

        self.federal_range = re.compile(
            r'\b\d{1,3}\s+U\.S\.C\.\s+§§\s*\d+(?:\s*[-–]\s*\d+)(?:\s*,\s*\d+(?:\s*[-–]\s*\d+))*\s*\(\d{4}\)?',
            re.IGNORECASE
        )

        # State Code Pattern
        self.state_code = re.compile(
            r'\b(?:Ala\.|Alaska|Ariz\.|Ark\.|Cal\.|Colo\.|Conn\.|Del\.|Fla\.|Ga\.|Haw\.|Idaho|Ill\.|Ind\.|Iowa|Kan\.|Ky\.|La\.|Me\.|Md\.|Mass\.|Mich\.|Minn\.|Miss\.|Mo\.|Mont\.|Neb\.|Nev\.|N\.H\.|N\.J\.|N\.M\.|N\.Y\.|N\.C\.|N\.D\.|Ohio|Okla\.|Or\.|Pa\.|R\.I\.|S\.C\.|S\.D\.|Tenn\.|Tex\.|Utah|Vt\.|Va\.|Wash\.|W\.Va\.|Wis\.|Wyo\.)\s+(?:Code|Rev\.|Rev\.\s+Stat\.|Compiled\s+Stat\.|Penal\s+Code|Bus\.\s+&\s+Prof\.\s+Code|Fam\.\s+Code|Prob\.\s+Code|Veh\.\s+Code)(?:\s+Ann\.)?\.?\s+§+\s*[\d\.]+(?:\([a-z]\))?(?:\s*\(\d{4}\))?',
            re.IGNORECASE
        )

        # Internal Revenue Code
        self.irc = re.compile(
            r'\bI\.R\.C\.\s+§+\s*\d+(?:\([a-z]\))?(?:\([A-Z]\))?(?:\([A-Z]\))?',
            re.IGNORECASE
        )

        # Session Laws
        self.session_law = re.compile(
            r'\b(?:[A-Z][a-zA-Z\s&]+Act)\s+(?:of\s+)?(?:Pub\.\s+L\.\s+No\.|Ch\.)\s+\d{3}[-–]\d{1,4}\s*,\s*\d+\s+(?:Stat\.|[A-Z]{2}\s+Stat\.)\s+\d+\s*\(\d{4}\)',
            re.IGNORECASE
        )

        # Federal Rules
        self.fed_rules_civ = re.compile(
            r'\bFed\.\s+R\.\s+Civ\.\s+P\.\s+\d+(?:\([a-z]\))?(?:\([A-Z]\))?',
            re.IGNORECASE
        )

        self.fed_rules_crim = re.compile(
            r'\bFed\.\s+R\.\s+Crim\.\s+P\.\s+\d+(?:\([a-z]\))?',
            re.IGNORECASE
        )

        self.fed_rules_evid = re.compile(
            r'\bFed\.\s+R\.\s+Evid\.\s+\d+(?:\([a-z]\))?(?:\([A-Z]\))?',
            re.IGNORECASE
        )

        # Ordinances
        self.ordinance = re.compile(
            r'\b(?:[A-Z][a-z]+),\s+(?:Ill|N\.Y\.|Tex\.|Cal\.|Ohio|Fla\.)\s*,\s+(?:Municipal|City|County)\s+Code\s+§+\s*[\d\-\.]+\s*\(\d{4}\)',
            re.IGNORECASE
        )

        # Restatements
        self.restatement = re.compile(
            r'\bRestatement\s+\((?:First|Second|Third|Fourth)\)\s+of\s+(?:[A-Za-z\s&]+)\s+§+\s*\d+\s*\((?:Am\.\s+Law\s+Inst\.|American\s+Law\s+Institute)\s+\d{4}\)',
            re.IGNORECASE
        )

    def validate_federal_code(self, citation):
        """Validate federal U.S.C. citation"""
        return bool(self.federal_usc.search(citation) or self.federal_usca.search(citation))

    def validate_state_code(self, citation):
        """Validate state code citation"""
        return bool(self.state_code.search(citation))

    def validate_internal_revenue_code(self, citation):
        """Validate I.R.C. citation"""
        return bool(self.irc.search(citation))

    def validate_federal_rules(self, citation):
        """Validate Federal Rules citation"""
        return bool(
            self.fed_rules_civ.search(citation) or
            self.fed_rules_crim.search(citation) or
            self.fed_rules_evid.search(citation)
        )

    def validate_session_law(self, citation):
        """Validate session law citation"""
        return bool(self.session_law.search(citation))

    def validate_ordinance(self, citation):
        """Validate municipal ordinance citation"""
        return bool(self.ordinance.search(citation))

    def validate_restatement(self, citation):
        """Validate Restatement citation"""
        return bool(self.restatement.search(citation))

    def extract_all_citations(self, text):
        """Extract all statute citations from text"""
        citations = {
            'federal_codes': self.federal_usc.findall(text),
            'state_codes': self.state_code.findall(text),
            'irc': self.irc.findall(text),
            'federal_rules': (
                self.fed_rules_civ.findall(text) +
                self.fed_rules_crim.findall(text) +
                self.fed_rules_evid.findall(text)
            ),
            'session_laws': self.session_law.findall(text),
            'ordinances': self.ordinance.findall(text),
            'restatements': self.restatement.findall(text),
        }
        return citations


# Usage Example:
validator = StatuteCitationValidator()

# Test citations
test_citations = [
    "42 U.S.C. § 1983 (2018)",  # Valid
    "Cal. Penal Code § 187 (West 2018)",  # Valid
    "Fed. R. Civ. P. 56",  # Valid
    "I.R.C. § 61",  # Valid
    "Civil Rights Act of 1964, Pub. L. No. 88-352, 78 Stat. 241 (1964)",  # Valid
    "Chicago, Ill., Municipal Code § 8-4-015 (2020)",  # Valid
    "42 USC 1983 (2018)",  # Invalid (missing section symbol)
    "Cal Penal Code 187 (2018)",  # Invalid (missing proper abbreviation and symbol)
]

for citation in test_citations:
    is_valid = (
        validator.validate_federal_code(citation) or
        validator.validate_state_code(citation) or
        validator.validate_federal_rules(citation) or
        validator.validate_internal_revenue_code(citation) or
        validator.validate_session_law(citation) or
        validator.validate_ordinance(citation) or
        validator.validate_restatement(citation)
    )
    print(f"{'✓ VALID' if is_valid else '✗ INVALID'}: {citation}")
```

## Key Takeaways for Rule 12 Compliance

1. **Always include section symbol (§)** - Never omit this critical element
2. **Use proper abbreviations** - Verify against Bluebook Table T1
3. **Include jurisdiction for state codes** - State abbreviation is mandatory
4. **Cite official codes first** - U.S.C. for federal, state official codes when available
5. **Year requirements vary** - Current U.S.C. doesn't need year; state codes typically do
6. **Session laws require complete information** - Pub. L. No., volume, reporter, and page
7. **Rules citations don't need years** - Presumed current and updated by amendments
8. **Parentheticals for context** - Use explanatory parentheticals for clarity
9. **Consistency is key** - Maintain consistent formatting throughout document
10. **Check local rules** - Court-specific or jurisdiction-specific rules may vary
