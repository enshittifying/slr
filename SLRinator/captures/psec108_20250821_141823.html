6: Punctuation

6.108: Parentheses and brackets in mathematics

Chapter Contents / Brackets and Braces

Parentheses and brackets are used as common delimiters in mathematical expressions. The order, which extends to braces (see 6.111), is as follows: {[( )]}. In displayed expressions, the sequence can be extended when necessary by large parentheses, brackets, and braces (in that order).

  • { [ ( { [ ( ) ] } ) ] }

In running text, braces are sometimes omitted from this sequence. Mathematical angle brackets ⟨ ⟩, vertical bars | |, and double vertical bars || || carry special mathematical significance and should not be used to supplement the sequence of common delimiters.
