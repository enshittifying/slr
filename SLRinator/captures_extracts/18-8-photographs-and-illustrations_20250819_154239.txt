Emojis may be cited by a replication of the emoji itself (if possible), a parenthetical noting the official name or description of the emoji (often indicated as the CLDR Short Name by Unicode), and the Unicode number associated with the emoji. If the emoji does not have an official name or description, include the best description possible. For emojis with multiple Unicode numbers, attempt to use the number associated with the variation being cited; if that is not possible, any code number that would direct to the emoji itself will suffice:

When an emoji does not have a Unicode number associated with it, cite it as an image.
For purposes of any rule that might require the inclusion of an emoji, emojis should be retained when possible. Where an emoji cannot be retained due to an inability to replicate it in the text of a citation, this should be indicated in a parenthetical noting the omission and the Unicode number of the omitted emojis.
