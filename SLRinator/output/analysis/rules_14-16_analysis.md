## Rule 14: Administrative and Executive Materials

### Overview
Rule 14 covers administrative materials including rules, regulations, agency decisions, and executive orders. While not primarily about books/articles/periodicals, understanding administrative citations is crucial for comprehensive legal research.

### 14.1 Basic Citation Forms

**Regulations (Codified):**
- Format: `[Volume] C.F.R. § [section] ([Year])`
- Example: `40 C.F.R. § 261.4 (2020)`
- Use current edition year of CFR

**Regulations (Not Yet Codified):**
- Format: `[Volume] Fed. Reg. [page] ([Date])`
- Example: `85 Fed. Reg. 12345 (Mar. 3, 2020)`

**Agency Decisions:**
- Similar to case citations
- Include agency name if not evident from reporter
- Example: `In re XYZ Corp., 123 F.T.C. 456 (1984)`

**Executive Orders:**
- Format: `Exec. Order No. [number], [volume] Fed. Reg. [page] ([year])`
- Example: `Exec. Order No. 13,771, 82 Fed. Reg. 9339 (2017)`

### 14.5 Short Forms for Regulations

**Using "Id.":**
- Only if immediately preceding citation is identical
- Example: `Id.`

**Shortened Form:**
- Use section number if context clear
- Example: After full cite to `40 C.F.R. § 261.4 (2020)`, later cite as `§ 261.4`

**Supra Note:**
- Can be used for regulations
- Example: `40 C.F.R. § 261.4, supra note 5`

## Rule 15: Books, Reports, and Other Nonperiodic Materials

### Overview
Basic format: `Author, Title (edition year)`

### 15.1 Author Names

**Single Author:**
- Include full name as appears on title page
- Example: `John Calvin Jeffries, Jr.`

**Two Authors:**
- List both with ampersand (&)
- Example: `John Smith & Jane Doe`

**Three or More Authors:**
- Use first author followed by "et al."
- Example: `John Smith et al.`
- Academic writing convention

**Institutional Authors:**
- Use full name as given
- Abbreviate according to Bluebook rules
- Example: `U.S. Environmental Protection Agency`
- Well-known acronyms: `EPA` (if appropriate)

**No Author:**
- Begin with title
- Or indicate institutional author if applicable

### 15.2 Editor or Translator

**Works by Editor (no author):**
- Format: `Title (Editor Name ed., Year)`
- Example: `Commentaries on the Laws of England (John Doe ed., 2020)`

**Multiple Editors:**
- Use "eds." plural
- Example: `(John Doe & Jane Smith eds., 2020)`

**Translator:**
- Use "trans." abbreviation
- Example: `(Jane Doe trans., 2020)`

### 15.3 Title

**Italicization:**
- All book titles must be italicized
- Not in quotation marks

**Capitalization:**
- Capitalize each word EXCEPT:
  - Articles (a, an, the)
  - Conjunctions (and, but, or)
  - Prepositions (of, in, on, at, to)
- ALWAYS capitalize if:
  - First word of title
  - First word of subtitle
  - Proper nouns

**Examples:**
- ✓ `The Death of Contract`
- ✓ `Law and Economics: A Comparative Approach to Theory and Practice`
- ✗ `The death of contract` (incorrect - "Death" should be capitalized)
- ✗ `Law And Economics` (incorrect - "And" should be lowercase)

**Subtitles:**
- Include after colon
- Example: `Title: Subtitle`

### 15.4 Edition, Publisher, and Date

**Edition Numbers:**
- Omit if first edition
- Include if 2d, 3d, 4th, etc.
- Use superscript if possible
- Example: `(2d ed. 2020)` or `(2d ed. 2020)`

**Publisher:**
- Generally not required for law review citations
- Required for:
  - Lesser-known works
  - Specific editions of codes
  - Multi-volume works when clarity needed

**Year:**
- ALWAYS include year of publication
- Use year of edition cited
- Example: `(2020)` or `(3d ed. 2019)`

### 15.5 Shorter Works in Collection

**15.5.1 Essays in Edited Collections:**

Format: `Author, Title of Essay, in Book Title page, pinpoint (Editor ed., edition year)`

Example:
```
John Doe, Essay Title, in Book Title 123, 130 (Jane Smith ed., 2d ed. 2020)
```

Components:
- `123` = starting page of essay
- `130` = pinpoint citation
- Multiple editors: use `eds.`

**15.5.2 Collected Documents:**

Format: `Document Description, in Book Title page (Editor ed., Year)`

Example:
```
Letter from Thomas Jefferson to John Adams (Dec. 1, 1800), in 5 Collected Letters of Thomas Jefferson 67 (Julian Boyd ed., 2000)
```

### 15.6 Prefaces, Forewords, Introductions, Epilogues

Format: `Author, Introduction, in Author, Book Title at page (Year)`

Example:
```
Jane Doe, Introduction, in John Smith, Book Title at xiii (2020)
```

If introduction has its own title:
```
Jane Doe, Title of Introduction, in John Smith, Book Title at xiii (2020)
```

### 15.7 Serial Numbers

For reports or books in a series:

Example:
```
Author, Title, Univ. of X Working Paper No. 123 (2020)
```

UN Documents:
```
U.N. Doc. A/54/PV.23 (1999)
```

### 15.8 Special Citation Forms

**Treatises and Multi-Volume Works:**

Format: `Volume Author et al., Title § section (edition year)`

Example:
```
2 Charles Alan Wright et al., Federal Practice & Procedure § 1234 (3d ed. 2019)
```

**Looseleaf Services:**
- Include date of last update
- Example: `5 CCH Consumer Finance Guide ¶ 72,205 (2019)`

### 15.9 Electronic Sources

**Books Accessed Online:**
- Prefer print citation format
- If online-only, add URL

Format: `Author, Title (Year), available at URL`

Example:
```
John Doe, Book Title (2020), available at https://example.com/book.pdf
```

### 15.10 Short Citation Forms

**Using "Id.":**
- If immediately preceding citation is the same source
- Example: `Id. at 100`

**Using "supra":**
- Format: `Author, supra note 5, at 100`
- Example: `Doe, supra note 5, at 100`

**Multiple works by same author:**
- Include short title
- Example: `Doe, Short Title, supra note 5, at 100`

**Hereinafter:**
- Use for long titles
- Provide in first full citation
- Example: `... (hereinafter "Short Title"). Then: Short Title, supra note 5, at 100`

## Rule 16: Periodical Materials

### Overview

Covers law review articles, journal articles, magazine articles, and newspaper articles.

Basic format: `Author, Title, Volume Periodical Name Page (Year)`

### 16.1 Basic Citation Forms

**Academic Journals (Law Reviews):**
```
Author, Title, Volume Journal Abbreviation Page (Year)
```

Example:
```
John Doe, Article Title, 123 Harv. L. Rev. 456 (2020)
```

**Magazines:**
```
Author, Title, Magazine Name, Month Day, Year, at page
```

Example:
```
Jane Smith, Article, Time, Jan. 1, 2020, at 20
```

**Newspapers:**
```
Author, Title, Newspaper Name, Date, at page
```

Example:
```
John Doe, Article, N.Y. Times, Jan. 5, 2021, at A1
```

### 16.2 Author Names

**Single Author:**
- List as appears in publication
- Full name preferred
- Example: `John Calvin Jeffries, Jr.`

**Two Authors:**
- List both with ampersand
- Example: `John Smith & Jane Doe`

**Three or More Authors:**
- First author followed by "et al."
- Example: `John Smith et al.`

**Unsigned Articles:**
- If institutional author: use it
- If student note without author: begin with title
- Can indicate "Note" or "Comment" as part of title

### 16.3 Title Treatment

**Academic Writing (Law Reviews):**
- Article titles in ITALICS
- Capitalize properly (same rules as books)

**Court Documents:**
- Article titles in regular type within quotation marks
- Different convention from academic writing

**Capitalization Rules:**
- Capitalize all words EXCEPT:
  - Articles (a, an, the)
  - Conjunctions (and, but, or, nor)
  - Prepositions of 4 letters or fewer (of, in, on, at, to, from, with)
- ALWAYS capitalize:
  - First word
  - Last word
  - First word after colon (subtitle)
  - All other words

**Include Punctuation:**
- Keep question marks, exclamation points
- Example: `What Is Law?` or `The End of History?`

### 16.4 Consecutively Paginated Journals

**Most law reviews are consecutively paginated:**
- Volume starts at page 1
- Continues through all issues
- No issue number needed

**Citation Format:**
```
Author, Title, Volume Journal Page (Year)
```

Example:
```
Jane Doe, Article, 50 Stan. L. Rev. 1234 (2020)
```

**No need for:**
- Issue number
- Month
- Season

### 16.5 Nonconsecutively Paginated Journals and Magazines

**Each issue starts at page 1:**
- MUST include issue information
- Include issue number OR date

**Format with Issue Number:**
```
Author, Title, Volume Journal, no. Issue, at page (Year)
```

Example:
```
John Smith, Article, 23 Nat'l L.J., no. 5, at 12 (2020)
```

**Format with Date:**
```
Author, Title, Magazine, Month Day, Year, at page
```

Example:
```
Jane Doe, Article, The Economist, Mar. 3, 2021, at 54
```

**Academic Journals with Issue Numbers:**
```
Author, Title, Volume(Issue) Journal Page (Year)
```

Example:
```
John Smith, Article, 4(3) Int'l J. of Law 15 (2020)
```
(Volume 4, Issue 3, page 15)

### 16.6 Newspapers

**Full Citation Format:**
```
Author, Title, Newspaper Name, Date, at page
```

**Components:**

1. **Author:**
   - If byline exists, use it
   - If no author, begin with title

2. **Title:**
   - Article title (italicized in law reviews)
   - Capitalize properly

3. **Newspaper Name:**
   - Italicized in law reviews
   - Use abbreviations per Bluebook
   - Example: `N.Y. Times` not `New York Times`

4. **Date:**
   - Full date: `Month Day, Year`
   - Example: `Jan. 5, 2021`

5. **Page:**
   - Section and page
   - Example: `at A1` (Section A, page 1)

**Examples:**
```
John Doe, Article Title, N.Y. Times, Jan. 5, 2021, at A1
```

```
Article Title, Wash. Post, Dec. 1, 2020, at C3
```
(No author - starts with title)

**Editions:**
- If relevant, note in parenthetical
- Example: `... at A1 (late ed.)`

**Online Newspapers:**
- Cite to print if available
- If online-only, include URL
- See Rule 18 for internet sources

### 16.7 Special Citation Forms

**16.7.1 Student-Written Law Review Materials:**

**Student Notes/Comments:**
- Can indicate "Note" or "Comment" before title
- If author known, cite normally
- If unsigned, treat designation as part of title

Examples:
```
Note, Title of Student Note, 130 Harv. L. Rev. 2345 (2017)
```

```
Sarah Johnson, Comment, Title, 45 Yale L.J. 678 (2020)
```

**16.7.2 Book Reviews:**

Format: `Author, Title of Review, Volume Journal Page (Year) (reviewing Book Citation)`

Example:
```
Jane Doe, Title of Review, 130 Harv. L. Rev. 123 (2017) (reviewing John Smith, Book Title (2016))
```

**16.7.3 Symposia and Colloquia:**

**Optional Parenthetical:**
- Can note symposium if relevant
- Not required unless needed for clarity

Example:
```
John Doe, Article, 100 Colum. L. Rev. 1 (2000) (Symposium: Law and Technology)
```

**16.7.4 Commentaries and Special Designations:**

Journals may have sections like:
- Notes
- Comments
- Essays
- Developments
- Recent Cases

Cite with whatever title is given. Add parenthetical if needed for clarity.

**16.7.5 Multipart Articles:**

**Published in multiple parts:**
- Cite each part separately
- Or cite first part with explanatory parenthetical

Example:
```
John Doe, Article Title (Part I), 50 Yale L.J. 1 (2020)
John Doe, Article Title (Part II), 50 Yale L.J. 201 (2020)
```

**16.7.6 Annotations:**

**A.L.R. Annotations:**

Format: `Author, Annotation, Topic, Volume A.L.R. Series Page (Year)`

Example:
```
John Doe, Annotation, Validity of State Legislation, 12 A.L.R.5th 123 (1993)
```

**16.7.7 Proceedings and ABA Section Reports:**

**Institute Proceedings:**

Format: `Proceedings of [Institute] Year, at page (publication year)`

Example:
```
Proceedings of the ABA Section of Antitrust Law 2018, at 45 (2019)
```

**16.7.8 Newsletters:**

**Organizational Newsletters:**

Format: `Organization, Newsletter Title, Date, at page`

Example:
```
ACLU Privacy Newsletter, Jan. 2020, at 2
```

If not widely available: Add `(on file with author)`

### 16.8 Electronic Sources

**Online Access:**
- Cite to print format when possible
- If online-only, include URL

**Online Journals (ahead of print):**
- Use DOI if available
- Or indicate "forthcoming"

**Blog Posts:**

Format: `Author, Title, Blog Name (Date, Time), URL`

Example:
```
Jane Doe, Blog Post Title, LAW BLOG (Jan. 2, 2020, 10:00 AM), https://lawblog.com/post
```

### 16.9 Short Citation Forms

**Using "Id.":**
- If immediately preceding citation is same article
- Different pinpoint allowed

Example:
```
Full cite: John Doe, Article, 123 Harv. L. Rev. 456 (2020)
Short: Id. at 460
```

**Using "supra":**

Format: `Author, supra note #, at page`

Example:
```
Full cite in note 4: John Doe, Article, 123 Harv. L. Rev. 456 (2020)
Short in note 10: Doe, supra note 4, at 460
```

**Multiple works by same author:**
- Include short title

Example:
```
Doe, Short Title, supra note 4, at 460
```

**Important:**
- "Supra" allowed for articles
- NOT allowed for cases and most primary sources

## Author Name Formatting

### General Principles

**As Published:**
- Use author's name exactly as appears on title page or byline
- Preserve:
  - Middle initials
  - Jr., Sr., III, etc.
  - Hyphens in surnames
  - Unusual spacing or capitalization (if intentional)

### Specific Formatting Rules

**Suffixes:**
```
✓ John Smith, Jr.
✓ John Smith III
✓ John Smith Sr.
```

**Middle Names/Initials:**
```
✓ John Calvin Jeffries, Jr.
✓ John C. Jeffries, Jr.
✓ John Jeffries
```

**Hyphenated Names:**
```
✓ Mary Smith-Jones
✓ Jean-Paul Sartre
```

**Single Name Authors:**
```
✓ Voltaire
✓ Blackstone (if that's how published)
```

### Multiple Authors

**Two Authors:**
```
Format: FirstAuthor & SecondAuthor

✓ John Smith & Jane Doe
✓ John C. Jeffries, Jr. & Paul B. Stephan III
```

**Three or More Authors (Academic Writing):**
```
Format: FirstAuthor et al.

✓ John Smith et al.
✗ John Smith, Jane Doe, & Robert Jones (too many - use et al.)
```

**"Et al." Formatting:**
- Lowercase "et"
- Period after "al"
- Space between "et" and "al"
- Not italicized in citations

### Institutional Authors

**Government Agencies:**
```
✓ U.S. Environmental Protection Agency
✓ EPA (if using acronym)
✓ U.S. Food & Drug Admin.
```

**Organizations:**
```
✓ American Law Institute
✓ A.L.I. (abbreviated)
✓ American Bar Association
✓ A.B.A. (abbreviated)
```

**Corporate Authors:**
- Use as appears on title page
- Abbreviate "Inc." "Corp." "Ltd." per Bluebook tables

### Editors and Translators

**As Editors:**
```
Format: (Editor Name ed., Year)

Single: (John Doe ed., 2020)
Multiple: (John Doe & Jane Smith eds., 2020)
Multiple (3+): (John Doe et al. eds., 2020)
```

**As Translators:**
```
Format: (Translator Name trans., Year)

(Jane Doe trans., 2020)
```

**Both Editor and Translator:**
```
(John Doe ed. & Jane Smith trans., 2020)
```

## Title Treatment

### Books (Rule 15.3)

**Formatting:**
- ALWAYS italicized
- NEVER in quotation marks
- NEVER in plain text

**Capitalization:**

Capitalize:
- ✓ First word
- ✓ Last word
- ✓ All nouns, pronouns, verbs, adjectives, adverbs
- ✓ All words of 5+ letters

Do NOT capitalize (unless first/last word):
- ✗ Articles: a, an, the
- ✗ Conjunctions: and, but, or, nor, yet, so
- ✗ Prepositions of 4 letters or fewer: of, in, on, at, to, for, from, with, by

**Examples:**
```
✓ The Death of Contract
✓ Law and Economics: A Comparative Approach
✓ Commentaries on the Laws of England
✓ The Common Law
✓ What Not to Do

✗ The Death Of Contract (wrong - "Of" should be lowercase)
✗ Law And Economics (wrong - "And" should be lowercase)
✗ the common law (wrong - "The" should be capitalized)
```

**Subtitles:**
- Separated by colon
- First word of subtitle always capitalized
- Same capitalization rules apply to subtitle

```
✓ Title: A Subtitle Here
✓ Law: An Analysis of Contemporary Practice
```

### Articles in Books (Rule 15.5)

**Essay/Chapter Titles:**
- NOT italicized
- Not in quotation marks in Bluebook style
- Same capitalization rules as books

**Book Title:**
- Preceded by "in"
- Italicized

**Example:**
```
John Doe, Essay on Constitutional Law, in Book of Essays 123 (Jane Smith ed., 2020)
         ^^^ NOT italicized ^^^         ^^ italicized ^^
```

### Periodical Articles (Rule 16.3)

**Academic Writing (Law Reviews):**
- Article titles in ITALICS
- NOT in quotation marks
- Same capitalization rules

**Court Documents/Practitioner Writing:**
- Article titles in quotation marks
- NOT italicized
- Regular roman type

**Examples (Law Review Style):**
```
✓ John Doe, The Future of Constitutional Law, 100 Yale L.J. 1 (2020)
           ^^^ italicized ^^^

✓ Jane Smith, What Is Privacy?, 50 Harv. L. Rev. 234 (2019)
             ^^ italicized (including ?) ^^
```

**Examples (Court Document Style):**
```
✓ John Doe, "The Future of Constitutional Law," 100 Yale L.J. 1 (2020)
           ^^^ in quotation marks ^^^
```

### Newspaper and Magazine Articles (Rules 16.5-16.6)

**Titles:**
- Italicized in academic writing
- Same capitalization rules

**Publication Names:**
- ALWAYS italicized
- Use abbreviations per Bluebook tables

**Examples:**
```
✓ Article Title, N.Y. Times, Jan. 1, 2020, at A1
  ^^^^^ italicized  ^^^^^^^^ italicized

✓ Jane Doe, Magazine Article, Time, Mar. 2021, at 15
           ^^^^^^^ italicized  ^^^^ italicized
```

### Special Considerations

**Question Marks and Exclamation Points:**
- Include in title
- Part of title formatting

```
✓ What Is Law?
✓ The Revolution Begins!
```

**Quotations in Titles:**
- Maintain original punctuation
- May require nested quotes

```
✓ "The Death of the 'Living Constitution'"
```

**All Caps in Original:**
- Convert to standard capitalization
- Follow Bluebook rules

```
Original: THE DEATH OF CONTRACT
Cite as: The Death of Contract
```

**Acronyms in Titles:**
- Keep in all caps
- Example: `The Role of the EPA in Environmental Law`

**Foreign Journals:**
- Use original title abbreviation per T13 for that language
- May include translation in parenthetical
- Example: `20 Revue Internationale de Droit Comparé 25 (1988) (Fr.) ("International Review of Comparative Law")`

## Pinpoint Citations

### Purpose

**Pinpoint citations** (also called "jump citations" or "pincites") direct readers to the specific page or location where cited material appears.

### Basic Format

**General Rule:**
- Provide first page of source
- Then provide specific page(s) referenced

**Format:** `[Volume] [Source] [First Page], [Pinpoint] ([Year])`

### Books

**Single Pinpoint:**
```
Format: Author, Title page (Year)

John Doe, Book Title 45 (2020)
                     ^^ pinpoint
```

**Page Range:**
```
John Doe, Book Title 45-47 (2020)
                     ^^^^^ pages 45 through 47
```

**Multiple Nonconsecutive Pages:**
```
John Doe, Book Title 45, 67, 89 (2020)
                     ^^^^^^^^^^ separate pages
```

### Articles

**Basic Format:**
```
Author, Title, Volume Journal First Page, Pinpoint (Year)

John Doe, Article, 123 Harv. L. Rev. 456, 460 (2020)
                                     ^^^  ^^^ pinpoint
                                   first  specific
```

**Multiple Pinpoints:**
```
John Doe, Article, 123 Harv. L. Rev. 456, 460-62 (2020)
                                          ^^^^^^ range

John Doe, Article, 123 Harv. L. Rev. 456, 460, 465 (2020)
                                          ^^^^^^^^^^ separate
```

### Short Form Pinpoints

**Using "Id.":**
```
Full: John Doe, Article, 123 Harv. L. Rev. 456 (2020)
Id.: Id. at 460
        ^^ pinpoint

Id. at 460-61
    ^^ different pinpoint
```

**Using "supra":**
```
Full in note 5: John Doe, Article, 123 Harv. L. Rev. 456 (2020)
Short in note 15: Doe, supra note 5, at 460
                                     ^^ pinpoint
```

### Footnotes and Endnotes

**Citing to Footnote:**

Format: `at [page] n.[note number]`

```
John Doe, Article, 123 Harv. L. Rev. 456, 460 n.4 (2020)
                                          ^^^^^^ page 460, footnote 4

John Doe, Book Title 45 n.12 (2020)
                     ^^^^^^ page 45, footnote 12
```

**Multiple Footnotes:**
```
at 460 nn.4-5 (footnotes 4 and 5)
at 460 nn.4, 7 (footnotes 4 and 7)
```

**Footnotes Spanning Pages:**
```
at 460-61 n.4 (footnote 4 which spans pages 460-61)
```

### Special Materials

**Sections and Paragraphs:**
```
Author, Title § 15.3 (Year)
             ^^^^^^ section 15.3

Author, Article, 123 J. 456, 460 ¶ 15 (Year)
                             ^^^^^ paragraph 15
```

**Multiple Sections:**
```
§§ 15.3-.4 (sections 15.3 through 15.4)
§§ 15.3, 15.5 (sections 15.3 and 15.5)
```

**Graphical Materials:**
```
at 45 tbl.3 (table 3)
at 45 fig.2 (figure 2)
at 45 chart 1 (chart 1)
```

**Appendices:**
```
at 45 app. A (appendix A)
at app. B, 12 (appendix B, page 12)
```

**Prefatory Material:**
```
at xiii (roman numerals for preface/introduction pages)
at xv-xvi (range in prefatory material)
at 4 (Introduction) (specify if needed)
```

### Page Number Formatting

**Regular Pages:**
- Arabic numerals
- Example: `45`, `123`, `1234`

**Introductory Pages:**
- Roman numerals (lowercase)
- Example: `xiii`, `xv`, `xxiii`

**Ranges:**
- Use en-dash (–) or hyphen (-)
- Drop repetitious digits:
  - `45-47` not `45-47`
  - `100-05` not `100-105`
  - `1234-35` not `1234-1235`
  - BUT: `99-100` not `99-00`

**Newspapers:**
- Include section letter
- Example: `at A1`, `at C12`

### Common Pinpoint Errors

**ERROR: Missing pinpoint when quoting or referring to specific point**
```
✗ John Doe, Article, 123 Harv. L. Rev. 456 (2020)
  (quoting from specific page but no pinpoint)

✓ John Doe, Article, 123 Harv. L. Rev. 456, 460 (2020)
```

**ERROR: Incorrect "at" usage**
```
✗ John Doe, Article, 123 Harv. L. Rev. at 456 (2020)
  (no "at" before first page)

✓ John Doe, Article, 123 Harv. L. Rev. 456 (2020)
✓ Id. at 460 (but "at" in short forms)
```

**ERROR: Inconsistent page ranges**
```
✗ at 45-8 (ambiguous - is it 45-48 or 45-58?)

✓ at 45-48 (clear)
✓ at 145-48 (clear - drop "1" from second number)
```

## Forthcoming Publications

### Rule 17.3: Forthcoming Publications

**Definition:**
- Works accepted for publication but not yet published
- Different from unpublished manuscripts

### Journal Articles

**Format:** `Author, Title, [Volume] Journal (forthcoming Year)`

**With Volume Known:**
```
Jane Smith, Article Title, 123 Harv. L. Rev. (forthcoming 2022)
```

**Without Volume:**
```
Jane Smith, Article Title, Harv. L. Rev. (forthcoming 2022)
```

**Year Unknown:**
```
Jane Smith, Article Title, Harv. L. Rev. (forthcoming)
```

### Books

**Format:** `Author, Title (forthcoming Publisher Year)`

**Example:**
```
Jane Smith, Book Title (forthcoming Oxford Univ. Press 2021)
```

**If Year Unknown:**
```
Jane Smith, Book Title (forthcoming Oxford Univ. Press)
```

**If Publisher Unknown:**
```
Jane Smith, Book Title (forthcoming 2021)
```

### Distinguishing from Unpublished

**Forthcoming:**
- Accepted for publication
- Publisher/journal confirmed
- Use "forthcoming"

```
✓ John Doe, Article, 100 Yale L.J. (forthcoming 2022)
```

**Unpublished Manuscript:**
- Not yet accepted
- Under review or in draft
- Use "unpublished manuscript"

```
✓ John Doe, Article Draft (unpublished manuscript, 2021) (on file with author)
```

**Working Paper:**
- Preliminary draft
- May be in SSRN or repository
- Different citation form

```
✓ John Doe, Article Title, Harvard Law School Working Paper No. 123 (2021)
```

### Online Availability

**Forthcoming + Online Access:**

If forthcoming article available online (SSRN, repository):

```
Jane Smith, Article Title, 123 Harv. L. Rev. (forthcoming 2022), https://ssrn.com/abstract=123456
```

Or:
```
Jane Smith, Article Title, 123 Harv. L. Rev. (forthcoming 2022) (manuscript at 10), https://ssrn.com/abstract=123456
```

### Short Citation Forms

**Using "supra":**
```
Full: Jane Smith, Article, Harv. L. Rev. (forthcoming 2022)
Short: Smith, supra note 5 (manuscript at 10)
```

**Using "Id.":**
```
Id. (manuscript at 12)
```

### Common Scenarios

**Volume/Issue Not Assigned:**
```
Author, Title, Journal (forthcoming Year)
```

**Pagination Not Final:**
- Cannot cite to specific pages
- Use "(manuscript at X)" if citing draft

```
Smith, supra note 5 (manuscript at 10)
```

**After Publication:**
- Update citation to published version
- Include volume and page numbers
- Remove "forthcoming"

## Error Detection Patterns

### Author Name Errors

**Missing "et al." for 3+ Authors:**
```
✗ John Smith, Jane Doe, & Robert Jones, Article, 100 Yale L.J. 1 (2020)
✓ John Smith et al., Article, 100 Yale L.J. 1 (2020)
```

**Incorrect "et al." Formatting:**
```
✗ John Smith, et al.
✗ John Smith et. al.
✗ John Smith et al
✓ John Smith et al.
```

**Missing Ampersand for Two Authors:**
```
✗ John Smith, Jane Doe
✓ John Smith & Jane Doe
```

**Incorrect Suffix Formatting:**
```
✗ John Smith Jr.
✗ John Smith, Jr
✓ John Smith, Jr.
```

### Title Errors

**Capitalization Errors:**
```
✗ The death of contract
✗ Law And Economics
✗ the Common Law
✓ The Death of Contract
✓ Law and Economics
✓ The Common Law
```

**Missing Italics (Books):**
```
✗ John Doe, The Death of Contract (2020)
✓ John Doe, The Death of Contract (2020)
```

**Quotation Marks on Books (Wrong):**
```
✗ John Doe, "The Death of Contract" (2020)
✓ John Doe, The Death of Contract (2020)
```

**Wrong Article Title Format:**

Academic writing:
```
✗ John Doe, "Article Title", 100 Yale L.J. 1 (2020)
✓ John Doe, Article Title, 100 Yale L.J. 1 (2020)
```

Court documents:
```
✗ John Doe, Article Title, 100 Yale L.J. 1 (2020)
✓ John Doe, "Article Title," 100 Yale L.J. 1 (2020)
```

### Journal Name Errors

**Not Abbreviated:**
```
✗ John Doe, Article, 100 Harvard Law Review 1 (2020)
✓ John Doe, Article, 100 Harv. L. Rev. 1 (2020)
```

**Missing Small Caps:**
```
✗ John Doe, Article, 100 Harv. L. Rev. 1 (2020)
✓ John Doe, Article, 100 HARV. L. REV. 1 (2020)
```
(in law review footnotes)

**Incorrect Abbreviation:**
```
✗ John Doe, Article, 100 Harv. L.R. 1 (2020)
✓ John Doe, Article, 100 Harv. L. Rev. 1 (2020)
```

### Volume and Page Errors

**Missing First Page:**
```
✗ John Doe, Article, 100 Yale L.J. (2020)
✓ John Doe, Article, 100 Yale L.J. 1 (2020)
```

**Incorrect "at" Placement:**
```
✗ John Doe, Article, 100 Yale L.J. at 1 (2020)
✓ John Doe, Article, 100 Yale L.J. 1 (2020)
✓ Id. at 5 (but okay in short forms)
```

**Missing Issue Number (Nonconsecutive Pagination):**
```
✗ John Doe, Article, 23 Magazine 12 (2020)
✓ John Doe, Article, Magazine, Jan. 2020, at 12
```

**Pinpoint Without First Page:**
```
✗ John Doe, Article, 100 Yale L.J., 5 (2020)
✓ John Doe, Article, 100 Yale L.J. 1, 5 (2020)
```

### Date and Year Errors

**Missing Year:**
```
✗ John Doe, Book Title
✓ John Doe, Book Title (2020)
```

**Wrong Date Format (Newspapers):**
```
✗ John Doe, Article, N.Y. Times, 1/5/2021, at A1
✗ John Doe, Article, N.Y. Times, 2021-01-05, at A1
✓ John Doe, Article, N.Y. Times, Jan. 5, 2021, at A1
```

**Edition Without Year:**
```
✗ John Doe, Book Title (2d ed.)
✓ John Doe, Book Title (2d ed. 2020)
```

### Parenthetical Errors

**Missing Edition:**
```
✗ John Doe, Book Title (2020)
  (when citing 2d edition)
✓ John Doe, Book Title (2d ed. 2020)
```

**Wrong Edition Format:**
```
✗ John Doe, Book Title (2nd ed. 2020)
✗ John Doe, Book Title (second ed. 2020)
✓ John Doe, Book Title (2d ed. 2020)
```

**Missing Editor Notation:**
```
✗ Essay Title, in Book Title 45 (John Smith, 2020)
✓ Essay Title, in Book Title 45 (John Smith ed., 2020)
```

**Wrong Editor Plural:**
```
✗ Essay Title, in Book Title 45 (John Smith & Jane Doe ed., 2020)
✓ Essay Title, in Book Title 45 (John Smith & Jane Doe eds., 2020)
```

### Forthcoming Errors

**Using "Forthcoming" for Unpublished:**
```
✗ John Doe, Draft Article, 100 Yale L.J. (forthcoming 2020)
  (when not accepted for publication)
✓ John Doe, Draft Article (unpublished manuscript, 2020)
```

**Missing "Forthcoming":**
```
✗ Jane Smith, Article, 123 Harv. L. Rev. (2022)
  (when article is accepted but not yet published)
✓ Jane Smith, Article, 123 Harv. L. Rev. (forthcoming 2022)
```

### Short Form Errors

**Using "Supra" for Cases:**
```
✗ Roe v. Wade, supra note 5
✓ Roe, 410 U.S. at 116
```

**Missing "at" in Supra:**
```
✗ Doe, supra note 5, 460
✓ Doe, supra note 5, at 460
```

**Using "Id." for Non-Immediately Preceding:**
```
✗ Note 10: Smith article
  Note 11: Jones article
  Note 12: Id. at 45 (intending to cite Smith)

✓ Note 12: Smith, supra note 10, at 45
```

**Missing Note Reference:**
```
✗ Doe, supra at 460
✓ Doe, supra note 5, at 460
```

## Conclusion

This comprehensive analysis of Bluebook Rules 14-16 provides detailed guidance on citing:
- Administrative materials (Rule 14)
- Books and nonperiodic materials (Rule 15)
- Periodicals including law reviews, magazines, and newspapers (Rule 16)

Key takeaways:
1. **Author names**: Use full names, "et al." for 3+, ampersand for two
2. **Titles**: Italicize books and articles (in academic writing), proper capitalization
3. **Journal abbreviations**: Follow Table T13, use small caps in law review footnotes
4. **Pinpoint citations**: Essential for specific page references
5. **Forthcoming**: Use only when accepted for publication
6. **Error detection**: Watch for capitalization, formatting, abbreviation errors
