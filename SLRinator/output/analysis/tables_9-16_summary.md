## T9: Legislative Documents

### Overview
Abbreviations for U.S. federal legislative materials including bills, resolutions, reports, and documents.

**Total Entries:** 19

### Key Patterns

#### House vs. Senate Distinctions
- **House materials:** Begin with `H.R.` (note the period after both letters)
- **Senate materials:** Begin with `S.` (simple abbreviation)

#### Document Type Categories

**Bills:**
- House Bill: `H.R. 1234`
- Senate Bill: `S. 1234`

**Resolutions:**
- House Resolution: `H.R. Res. 456`
- Senate Resolution: `S. Res. 456`
- House Concurrent Resolution: `H.R. Con. Res. 78`
- Senate Concurrent Resolution: `S. Con. Res. 78`
- House Joint Resolution: `H.R.J. Res. 90` (note: no space before J)
- Senate Joint Resolution: `S.J. Res. 90` (note: no space after S.)

**Reports and Documents:**
- House Report: `H.R. Rep. No. 117-234`
- Senate Report: `S. Rep. No. 117-234`
- House Document: `H.R. Doc. No. 116-45`
- Senate Document: `S. Doc. No. 116-45`

**Special Materials:**
- Committee Print: `Comm. Print`
- Congressional Record: `165 Cong. Rec. 12345`
- Congressional Research Service: `Cong. Rsch. Serv.`

### Critical Rules

1. **Always include "No."** in report and document citations
2. **Format:** [Congress Number]-[Document Number] (e.g., `117-234`)
3. **"Treaty" is never abbreviated** in Senate Treaty Documents
4. **Spacing matters:** `H.R.J. Res.` has no space before J, but `H.R. Con. Res.` has spaces

### Common Errors

| Incorrect | Correct | Issue |
|-----------|---------|-------|
| `HR 1234` | `H.R. 1234` | Missing periods |
| `H. R. 1234` | `H.R. 1234` | Extra space |
| `H. Res. 456` | `H.R. Res. 456` | Missing R. |
| `S. J. Res. 90` | `S.J. Res. 90` | Extra space |
| `H.R. Report No. 117-1` | `H.R. Rep. No. 117-1` | Not abbreviated |
| `S. Tr. Doc. No. 115-1` | `S. Treaty Doc. No. 115-1` | Treaty incorrectly abbreviated |

## T16: Subdivisions

### Overview
Abbreviations for document parts and internal divisions.

**Total Entries:** 25

### CRITICAL SPACING RULE

**No space** between abbreviation and number for these FOUR:
- `fig.4` (figure 4)
- `n.12` (footnote 12)
- `hist. n.5` (historical note 5)
- `tbl.2` (table 2)

**Space required** for all others:
- `§ 1983` (section 1983)
- `art. II` (article II)

#### Document Parts

- volume(s): `vol.` - `vol. 5`
- division(s): `div.` - `div. A`

#### Reference Elements
- figure(s): `fig.` - `fig.4` (NO SPACE)
- table(s): `tbl.` - `tbl.2` (NO SPACE)
- footnote(s): `n.` - `n.12` (NO SPACE)
- historical note(s): `hist. n.` - `hist. n.5` (NO SPACE)
- line(s): `l.` - `l. 45`
- column(s): `col.` - `col. 3`
- number(s): `no.` - `No. 21-1234` (often capitalized)
- series: `ser.` - `ser. 2`

### Section Symbol Usage

**CRITICAL:** Always use § symbol, NEVER "sec."

- Single section: `§ 1983`
- Multiple sections: `§§ 101-105`
- Range: `§§ 1-10`
- Multiple non-consecutive: `§§ 100, 200, 300`

### Usage Examples

**Statutory Citations:**
```
42 U.S.C. § 1983 (2018)
Cal. Civ. Code §§ 1-10 (West 2020)
N.Y. Gen. Bus. Law § 349, subsec. (a) (McKinney 2024)
18 U.S.C. § 1001(a)(1) (parenthetical subsections)
```

**Constitutional Citations:**
```
U.S. CONST. art. I, § 8, cl. 3
U.S. CONST. amend. XIV, § 1
```

**Internal Cross-References:**
```
See supra Part II.A, para. 5.
See infra note 45 and accompanying text.
As shown in fig.4, the data...
The results in tbl.2 demonstrate...
```

### Critical Rules

1. **Never use "sec."** - always use § symbol
2. **Spacing exceptions:** Only fig., n., hist. n., tbl. have no space
3. **Double symbols for plurals:** §§ (not § plural)
4. **Lowercase generally:** except "No." in case numbers
5. **Combine hierarchically:** `§ 1001(a)(1)` uses parentheses, not "subsec."

### Common Errors

| Incorrect | Correct | Issue |
|-----------|---------|-------|
| `sec. 1983` | `§ 1983` | Must use § symbol |
| `§§ 1983` | `§ 1983` | Single section, not plural |
| `n. 12` | `n.12` | Extra space |
| `fig. 4` | `fig.4` | Extra space |
| `tbl. 2` | `tbl.2` | Extra space |
| `art.II` | `art. II` | Missing space |
| `subsection (a)` | `subsec. (a)` | Not abbreviated |

## Quick Reference Guide

### Abbreviation Decision Tree

```
1. Is it a MONTH?
   → May, June, July? → Never abbreviate
   → Other months → Abbreviate with period (Jan., Feb., etc.)

2. Is it a U.S. STATE?
   → Alaska, Hawaii, Idaho, Iowa, Ohio, Utah? → Never abbreviate
   → Other states → Use Bluebook abbreviation (Cal., N.Y., etc.)
   → NEVER use postal codes (CA, NY, etc.)

3. Is it a LEGISLATIVE DOCUMENT?
   → House material → Starts with H.R.
   → Senate material → Starts with S.
   → Include "No." in reports/documents

4. Is it a JUDICIAL TITLE?
   → Single judge/justice → J.
   → Multiple judges/justices → JJ.
   → Chief → C.J.
   → Attorney General → Att'y Gen. (with apostrophe)

5. Is it a DOCUMENT SUBDIVISION?
   → Section → § (never "sec.")
   → Footnote, figure, table, hist. note → NO SPACE before number
   → All others → Space before number

6. Is it a PERIODICAL?
   → Check T13 for institutional names
   → Check T14 for publishing terms
   → Combine: Harv. L. Rev., Colum. L. Rev.

7. Is it a SERVICE?
   → Abbreviate service name
   → Add (Publisher) in parentheses
   → Example: Bankr. L. Rep. (CCH)
```

### Top 20 Most Common Errors

1. Using postal codes instead of Bluebook state abbreviations
2. Abbreviating May, June, or July
3. Writing "sec." instead of §
4. Abbreviating Idaho, Alaska, Hawaii, Iowa, Ohio, or Utah
5. Adding space before numbers in n., fig., tbl., hist. n.
6. Omitting "No." in legislative reports/documents
7. Missing apostrophe in Att'y Gen. or Comm'r
8. Using single § for multiple sections (should be §§)
9. Wrong abbreviation for September (Sep. instead of Sept.)
10. Confusing H.R. (House Bill) with H.R. Res. (House Resolution)
11. Missing space in W. Va. (West Virginia)
12. Adding space in N.Y., N.J., etc.
13. Using Calif. outside of California Law Review context
14. Abbreviating "Treaty" in Senate Treaty Documents
15. Missing periods in abbreviations
16. Using "Js." for plural judges (should be JJ.)
17. Adding period after Comm'r (shouldn't have one)
18. Forgetting double 'm' in Comm'r
19. Using J. when JJ. is needed for multiple judges
20. Incorrect spacing in H.R.J. Res. or S.J. Res.

## Common Errors and Solutions

### State Abbreviations

**ERROR:** Using postal codes
```
WRONG: CA, NY, TX, FL
RIGHT: Cal., N.Y., Tex., Fla.
```

**ERROR:** Abbreviating non-abbreviatable states
```
WRONG: Id., Haw., Ak.
RIGHT: Idaho, Hawaii, Alaska
```

**ERROR:** Wrong spacing in two-letter abbreviations
```
WRONG: N. Y., S. C., N. J.
RIGHT: N.Y., S.C., N.J.
EXCEPTION: W. Va. (has space)
```

### Legislative Documents

**ERROR:** Missing periods
```
WRONG: HR 1234, S 456
RIGHT: H.R. 1234, S. 456
```

**ERROR:** Wrong spacing in joint resolutions
```
WRONG: H.R. J. Res. 90, S. J. Res. 90
RIGHT: H.R.J. Res. 90, S.J. Res. 90
```

**ERROR:** Missing "No." in reports
```
WRONG: H.R. Rep. 117-234
RIGHT: H.R. Rep. No. 117-234
```

### Judicial Titles

**ERROR:** Missing apostrophe
```
WRONG: Atty Gen., Commr
RIGHT: Att'y Gen., Comm'r
```

**ERROR:** Wrong plural
```
WRONG: Sotomayor and Kagan, J.
RIGHT: Sotomayor and Kagan, JJ.
```

**ERROR:** Extra period on Comm'r
```
WRONG: Comm'r.
RIGHT: Comm'r
```

### Months

**ERROR:** Abbreviating short months
```
WRONG: May., Jun., Jul.
RIGHT: May, June, July
```

**ERROR:** September too short
```
WRONG: Sep. 1, 2024
RIGHT: Sept. 1, 2024
```

### Subdivisions

**ERROR:** Using "sec." instead of §
```
WRONG: sec. 1983, section 1983
RIGHT: § 1983
```

**ERROR:** Wrong spacing
```
WRONG: n. 12, fig. 4, tbl. 2
RIGHT: n.12, fig.4, tbl.2

WRONG: art.II, ch.10, para.5
RIGHT: art. II, ch. 10, para. 5
```

**ERROR:** Wrong section symbol for plural
```
WRONG: § 101-105 (plural range)
RIGHT: §§ 101-105
```

## Best Practices for Cite Checkers

### 1. Context-Aware Checking
- California: Check if in "California Law Review" → use `Calif.`
- California: All other contexts → use `Cal.`
- "ed.": Could be "editor" or "edition" → context determines correctness

### 2. Hierarchical Validation
Apply tables in order:
1. T9-T11: Document and authority type
2. T10: Geographic location
3. T12: Dates
4. T13-T14: Periodical and publishing info
5. T15: Services
6. T16: Subdivisions and pinpoints

### 3. Common Pattern Recognition
Build regex libraries for frequent patterns:
- U.S. state: `\b[A-Z][a-z]*\.\b` or `\b(Alaska|Hawaii|Idaho|Iowa|Ohio|Utah)\b`
- Section cite: `§§?\s*\d+`
- Date: `(Jan\.|Feb\.|Mar\.|Apr\.|May|June|July|Aug\.|Sept\.|Oct\.|Nov\.|Dec\.)\s+\d{1,2},\s+\d{4}`
