1.6: Multiple Sources Following a Signal as a Verb

When multiple sources are attached to a textual signal, insert semicolons to separate them, add an "and" before the final source, and finish with a period. This rule applies even if only two sources are listed. If switching to a new signal, begin a new citation sentence. Example:

For a discussion of fun places to eat, see Example 1; Example 2; and Example 3. See also Example 4 (a particularly fun place to eat); Example 5 (same). For a discussion of the best theaters, see, for example, Example 6; and Example 7.
