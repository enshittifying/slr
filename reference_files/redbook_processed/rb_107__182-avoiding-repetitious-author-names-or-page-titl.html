18.2: Avoiding Repetitious Author Names or Page Titles

If there would be repetition between the author name, the page title, and/or the main page title, the general rule is to leave the main page title and any other nonrepetitive information. Two common scenarios:

Institutional author clear from main page title: If the institutional author is the same as the owner of the main page and the owner of the main page is clear from the main page title, omit the name of the institutional author and retain the main page title in small caps. See BB 18.2.2(a). Examples:

Correct: Directors, Then and Now, FBI, https://perma.cc/C4YL-47JD (archived July 7, 2017).
Incorrect: FBI, Directors, Then and Now, FBI, https://perma.cc/C4YL-47JD (archived July 7, 2017). See BB 18.2.2(b)(i). Examples:
Correct: See Fed. Jud. Ctr., https://perma.cc/524A-BHQD (archived July 7, 2017).
Incorrect: See Federal Judicial Center, Fed. Jud. Ctr., https://perma.cc/524A-BHQD (archived July 7, 2017). Incorrect: See Homepage, Fed. Jud. Ctr., https://perma.cc/524A-BHQD (archived July 7, 2017).
