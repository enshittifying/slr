18.6: Location Parentheticals for Websites

If a source is hard to find once you visit the URL in the citation, please insert a “to locate” parenthetical to help direct the reader to the particular information being cited. See BB 18.2.2(d). Example:

Archived Public Data Files: FY 2013, U.S. Soc. Sec. Admin., https://perma.cc/Y39C-7P3R (archived July 7, 2017) (to locate, select “View the live page,” then select the “FY 2013” tab, follow the “ALJ Disposition Data” hyperlink, and select “September 2013”).
