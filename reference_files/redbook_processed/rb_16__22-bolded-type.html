2.2: Bolded Type

SLR only uses bolded type for Part-level headings (including introductions, conclusions, and appendices), table and figure headings, and row and column headings within tables.

Do not use bold for emphasis; use italics instead. If quoted material uses bold for emphasis, convert it to italics without indicating the alteration, see RB 5.4, or omit the emphasis per RB 1.13.
