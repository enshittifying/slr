12.7: Abbreviating Local Statutes or Ordinances

Cite local ordinances—even those available only on websites—according to BB 12.9.2. Abbreviate the political subdivision and state name per BB Table T10 and the code name per BB Table T6. Do not provide the URL. Example:

See, e.g., S.F., Cal., Pub. Works Code § 2307.6(a) (2018).
